# Student-Performance-Analysis-EDA-Statistical-Testing-

This project conducts an exploratory data analysis (EDA) on a student dataset to uncover patterns and relationships between demographic factors, study habits, and academic outcomes. Using Python (Pandas, Matplotlib, Seaborn, SciPy) in a Jupyter Notebook, the analysis includes:

Data loading and integration – reading the dataset and inspecting its structure, shape, and summary statistics.

Exploratory Data Analysis – visualizing distributions of scores (math, reading, writing) through histograms, box plots by gender and parental education, and a correlation heatmap to examine relationships among score variables.

Data cleaning – checking for missing values and duplicates to ensure data quality.

Statistical hypothesis testing – performing a t-test to compare math scores based on test preparation course completion, an ANOVA to assess differences in reading scores across parental education levels, and a chi‑squared test to examine the association between gender and test preparation course enrollment.

The findings highlight significant effects of test preparation and parental education on student performance, as well as a notable relationship between gender and course completion behavior. This project demonstrates proficiency in data wrangling, visualization, and inferential statistics using Python.
